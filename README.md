PyGoat GSoC Proposal – Muskan Varshney
Overview

This repository contains my proposal for contributing to PyGoat under Google Summer of Code (GSoC). The goal of my project is to enhance PyGoat’s functionality and learning experience by developing new features, improving documentation, and creating practical labs for web application security.

About Me

I am Muskan Varshney, a passionate developer and cybersecurity enthusiast. I have experience leading and developing web applications from scratch, including the Cryptool Web Application, a multi-service security platform. I am actively contributing to open-source projects and continuously learning Python, cloud technologies, and cybersecurity concepts.

Proposal Highlights
Develop new security labs for educational purposes.
Improve PyGoat’s existing lab structure and documentation.
Focus on user-friendly, interactive exercises to teach web security concepts effectively.
Ensure maintainability and scalability of the project.
